Limelight

A lightweight local database



Features:
  • Encryption - Use an encryption key, so even if someone has access to your database, they won't have the data!
  • Lightweight - Unlike some modules (cough Express), Limelight only has 1 dependency.
  • JavaScript Filters - Filters are VanillaJS functions, so they're powerful with unlimited potential (wow that was cringe)
  • Admin Panel - Limelight comes with a free admin panel, Sublime!
